## 📦 Troubleshooting performance

If you experience issues, try these steps:

* Run as Administrator: Right-click the app icon and select "Run as administrator." This gives the app permission to display over full-screen games.
* Check display mode: If the overlay does not show, try switching your game from "Exclusive Fullscreen" to "Windowed Borderless." This mode allows other windows to sit on top of the game.
* Update graphics drivers: Ensure your video card software is current.
* Adjust overlay settings: If the frame rate drops, reduce the complexity of the crosshair design or disable unnecessary visual effects in the app menu.

## 🛡 Security and safety

This project functions as an overlay. It reads no game data and sends no information to the game server. It creates a simple image layer on your monitor. Because the software does not interact with game memory, it remains safe for common use.
